tmux 入门指南

什么是 tmux？

tmux 是一个终端复用工具，它允许你在一个终端窗口中创建多个会话、窗口和面板，让你的终端使用体验更加高效和灵活。

为什么需要使用 tmux？

作为开发者，你是否遇到过以下问题：

  • 打开多个终端窗口，桌面变得混乱
  • 远程连接断开后，正在运行的任务丢失
  • 需要在多个终端之间快速切换
  • 希望在不同会话之间共享终端状态

tmux 正是为了解决这些问题而生的！

1. 创建新会话

2. 分离会话

按下 Ctrl+b 然后按 d，可以将当前会话分离到后台。

3. 查看所有会话

4. 重新连接会话

5. 切换会话

按下 Ctrl+b 然后按 s，可以列出所有会话并进行选择。

窗口和面板操作

窗口操作

  • Ctrl+b c：创建新窗口
  • Ctrl+b w：列出所有窗口
  • Ctrl+b 数字：切换到指定编号的窗口
  • Ctrl+b ,：重命名当前窗口
  • Ctrl+b &：关闭当前窗口

面板操作

  • Ctrl+b %：垂直分割面板
  • Ctrl+b "：水平分割面板
  • Ctrl+b 方向键：在面板之间移动
  • Ctrl+b z：最大化/还原当前面板
  • Ctrl+b x：关闭当前面板

常用快捷键总结

实际应用场景

场景一：远程开发

当你通过 SSH 连接到远程服务器时，使用 tmux 可以确保即使连接断开，你的工作也不会丢失。

场景二：多任务处理

在一个终端窗口中，你可以同时：

  • 运行开发服务器
  • 查看日志
  • 编辑代码
  • 执行测试

场景三：团队协作

通过 tmux 的共享会话功能，多个开发者可以同时查看和操作同一个终端，非常适合结对编程或技术支持。

进阶技巧

  1. 会话管理：使用 tmux new-session 创建带有初始窗口和面板的会话
  2. 窗口布局：使用 tmux select-layout 快速切换面板布局
  3. 复制模式：按下 Ctrl+b [ 进入复制模式，使用方向键选择文本，按下 Enter 复制
  4. 插件管理：使用 TPM（Tmux Plugin Manager）安装插件

总结

tmux 是一个功能强大的终端复用工具，它可以大大提高你的工作效率。虽然刚开始使用时可能需要记住一些快捷键，但一旦掌握，你会发现它是开发过程中的得力助手。

从今天开始，尝试在你的日常工作中使用 tmux，相信你会爱上它的！

提示：如果你是 tmux 新手，建议先从基本的会话管理和面板操作开始，逐渐熟悉各种快捷键和配置选项。随着使用的深入，你会发现 tmux 的更多强大功能。
